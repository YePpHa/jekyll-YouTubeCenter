Install

To install YouTube Center please select the correct version below.

Userscript

For Greasemonkey, Scriptish or Tampermonkey.

Install on Userscript

Chrome/Blink based browsers

For the Chrome browser and other Blink based browsers including Opera (v15 or newer).

Install on Chrome Help to install

Firefox

Native add-on for Firefox.

Install on Firefox Add-on page

Maxthon

The extension for the Maxthon browser.

Install on Maxthon Add-on page

Safari

For the Safari browser.

Install on Safari

Old Opera

For the older version of Opera (v14 or less).

Install on Opera

Current version: v{{ site.ytc_stable_version }}.

What's next?

After you have successfully installed YouTube Center you will have to go to YouTube where a dialog will appear which will show you how to access the settings. From there you can configure YouTube Center just how you want it.
